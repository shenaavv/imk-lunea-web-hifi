## 🔧 CUSTOMIZATION EXAMPLES

### Change Brand Colors
```css
/* In src/css/styles.css */
:root {
    --color-primary: #YOUR-COLOR;
    --color-primary-light: #YOUR-COLOR;
    /* ... etc */
}
```

### Change Website Title
```html
<!-- In index.html head -->
<title>Your Website Name</title>
```

### Change Hero Title
```html
<!-- In index.html -->
<h1 class="hero-title">Your Title Here</h1>
```

### Add New Products
```html
<!-- Copy product card and modify -->
<div class="product-card featured-card">
    <div class="product-image">
        <img src="YOUR-IMAGE" alt="Your Product">
    </div>
    <!-- ... -->
</div>
```
